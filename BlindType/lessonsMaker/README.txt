Управляется с помощью файла 'config.txt'

Содержит построчно:
words.txt - название файла со словарём
input_lat.txt - название файла ввода
lessons_lat - название папки вывода
5 - минимальное количество символов в отфильтрованных словах

Все файлы в текущем каталоге!

Файл ввода должен содердать аргументы в следующем формате:

[название_урока] [имя_папки_1]
[последовательность_символов] [название_упражнения] [имя_файла_1]
[последовательность_символов] [название_упражнения] [имя_файла_2]
[последовательность_символов] [название_упражнения] [имя_файла_3]
[название_урока] [имя_папки_2]
[последовательность_символов] [название_упражнения] [имя_файла_1]
......
[название_урока] [имя_папки_n]
[последовательность_символов] [название_упражнения] [имя_файла_1]
[последовательность_символов] [название_упражнения] [имя_файла_2]


Имена папок и файлов должны быть разными. 

На странице они будут отображены исходя из алфавитного порядка их имён

[название_урока] и [название_упражнения]  будут помещены в файл '$dirInfo' в соответствующих каталогах
[название_упражнения] - первой строкой в конечном файле

Содержимое выходной папки следует поместить в папку 'Data/lessons'
