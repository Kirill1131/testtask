

Для полноценной работы приложения необходим запуск на сервере с поддержкой php

Приложение содержит три страницы:
'main.html', 'test.html' работают без php
'exercise.html' использует php-скрипты из папки 'php' (из скрипта exercise.js отправляются запросы для получения данных из папки 'Data').
Так же, если нет поддержки php, можно открыть в браузере 'exerciseNONPHPDEMO.html'


Наполнение в разделе 'тестирование' генерируется на сторонних API:

Для русской клавиатуры - 'https://fish-text.ru/get?number=4'
Для латинской - 'https://baconipsum.com/api/?type=meat-and-filler&sentences=7'


Формулы:

кол-во знаков минуту = ([количество_правильно_введённых_символов]/[количество_секунд]) *60  ;
точность = (100/[символов_всего])*([символов_всего]-[количество_ошибок])

Дополнительно:

Наполнение в разделе 'практика' производится автоматически с помощью языков php и js из файлов, лежащих по адресу 
'Data/lessons/[layout_dir]/[less_dir]/[ex].txt',
где: 
	layout_dir - название папки с языковой раскладкой
		обязательно содержит файл '$dirInfo', в котором
			первая строка: название раскладки, как оно должно отображаться на странице
			вторая строка: символьный код клавиатуры, 
					которая будет задействована, 
					соответствующий названию файла для её инициализации в папке
					'Data/keyboards/..'
	less_dir - название папки
		обязательно содержит файл '$dirInfo', в котором
			первая строка: название урока, как оно должно отображаться на странице
	ex - название файла
			первая строка: название упражнения, как оно должно отображаться на странице
			следующие: слова, которые должны участвовать в упражнениии


Для наполнения в соответствующем разделе была реализована программа на языке c#, составляющая файлы (последовательности слов) по строке символов, которая содержит значения соответствующиx клавиш, задействованных в упражнении; используя построчно-записанные файлы - словари.
